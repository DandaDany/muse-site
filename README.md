# My Map 電影上映地圖

把「電影正在哪些影城上映、有哪些場次」整理成一張可互動的網頁地圖，方便快速找到離自己最近、還有場次的影城。

## 線上地圖

- 公開地圖：<https://dandadany.github.io/muse-site/>

在地圖上可以：

- 用下拉選單切換不同電影。
- 搜尋影城名稱。
- 依品牌、縣市篩選。
- 點選影城看當天所有場次時間與地址，並可一鍵開啟 Google 地圖導航。

地圖每天會自動更新當天的場次資料。

## 功能簡介

- 以台灣為中心的互動地圖，涵蓋各大連鎖與地方影城據點。
- 每個影城以品牌 logo 標示，場次越多的影城標記越大。
- 手機與桌機皆有對應的操作介面。

## 技術概觀

- 前端：Leaflet 靜態網頁（`web/`），透過 GeoJSON 載入影城與場次資料。
- 發佈：GitHub Pages + GitHub Actions 自動部署，每日自動更新。
- 後端控制面：Django（部署於 Render），用來管理追蹤中的電影。
